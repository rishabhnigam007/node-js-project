To create project using command line : npm init
then give package name, version, description, author,ok

start node server : npm start

install mongodb : npm install mongoose
install express : npm install express
install body-parser : npm install body-parser
install hbs(template engine for frontend) : npm install hbs
install nodemon (same as devtool) : npm install nodemon --save-dev (for only development environment)

set "start":nodemon app.js in script in package-json file
